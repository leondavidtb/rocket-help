## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/leondavidtb/rockethelp.git
```

Entre no diretório do projeto

```bash
  cd rockethelp
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  expo run:android
```
